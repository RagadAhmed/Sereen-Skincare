# SEREEN Skin Care Storefront

Static ecommerce storefront for the SEREEN Skin Care brand.

## Files

- `index.html` - storefront markup and Netlify order form
- `styles.css` - responsive brand styling
- `app.js` - product rendering, cart drawer, and local cart storage
- `assets/images/` - optimized brand and product images
- `netlify.toml` - Netlify publish settings, headers, and fallback route

## Deploy to Netlify

1. Drag this folder into Netlify Drop, or connect it as a Git repository.
2. Use these build settings:
   - Build command: leave blank
   - Publish directory: `.`
3. Netlify will detect the `sereen-orders` form after deployment.
